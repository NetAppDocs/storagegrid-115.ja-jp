= クラウドストレージプールの使用
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
クラウドストレージプールを使用して、StorageGRID オブジェクトをS3 GlacierやMicrosoft Azure BLOBストレージなどの外部ストレージに移動できます。オブジェクトをグリッドの外部に移動すると、低コストのストレージ階層を活用した長期間のアーカイブが可能になります。
