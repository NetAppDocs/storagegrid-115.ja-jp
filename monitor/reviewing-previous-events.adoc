= 以前のイベントを確認しています
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
以前のイベントメッセージのリストを生成して、過去に発生した問題を特定することができます。

. Support *>* Tools *>* Grid Topology *を選択します。
. [*_site *>] > [*_grid node_*] > * ssm * > * Events * > * Reports * ] を選択します。
. 「 * テキスト * 」を選択します。
+
「* Last Event *」属性は「Charts（チャート）」ビューには表示されません。

. 属性 * を * 最後のイベント * に変更します。
. 必要に応じて、 * クイッククエリ * の期間を選択します。
. [ 更新（ Update ） ] をクリックします。
+
