= [オブジェクト（Objects）]タブの表示
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Objects タブには、 S3 および Swift の取り込み速度と読み出し速度に関する情報が表示されます。

各ストレージノード、各サイト、およびグリッド全体のオブジェクトタブが表示されます。ストレージノードが対象である場合は、オブジェクト数や、メタデータクエリやバックグラウンド検証に関する情報も表示されます。
